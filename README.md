2017级项目实训成果展示
===========
《呦呦育儿》 -HTML5与移动互联网开发方向
-------------------
项目简介<br>
-------
呦呦育儿APP是一款帮助父母科学健康照顾宝贝的软件，软件的使用对象可以是爸爸、妈妈，也可以是宝贝。<br>
我们设计了亲子时光模块，宝贝可以在线培养学习能力，还提供艺术模块，宝贝可以在线听音乐。宝贝也可以独自或者和父母一起完成小游戏，锻炼思维，培养动手能力。<br>
爸爸妈妈可以在线学习护理知识，还可以晒娃，发布动态记录宝贝成长。
